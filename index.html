Features

  • Rich Components

    There are many components to use such as select, multi-select, carousel, full page scrolling, pull to refresh and so on.

  • Super Tiny Size

    2.24 kB gzipped

  • Simple API

    Takes about a minute to get start. Creating an instance of an object by new keyword:
    new AlloyTouch(option)

  • Smoothing Animation

    The tween parameters of motion is a time-consuming progress with iterative adjustment and modify. Bring the ultimate experience to the users.

  • Easy To Extend

    Highly abstract logic design allows you to easily touch many user feedback scenes, not just scrolling.

  • Excellent Contributors

    AlloyTeam will be the first time to respond to any of your questions, any comments and suggestions please let us know.
